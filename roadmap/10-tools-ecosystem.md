# Tools & Ecosystem

Summary:
Overview of common tools and platforms AI Engineers use: Hugging Face, OpenAI, Ollama, Google Gemini, Mistral, Cohere, and SDKs for inference and model management.

Key tools:
- Hugging Face Hub, Inference API, Transformers library
- OpenAI APIs (GPT family), Anthropic, Google Gemini
- Local runtimes: Ollama, llama.cpp, transformers.js
- Vector DB providers: Pinecone, Milvus, Weaviate

Backlinks:
- Source roadmap: https://roadmap.sh/ai-engineer
- Hugging Face: https://huggingface.co
- Ollama: https://ollama.com

Practical tip: evaluate cost, latency, and licensing when choosing model providers.
