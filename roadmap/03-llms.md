# Large Language Models (LLMs)

Summary:
Understand modern transformer-based LLMs, pre-trained models, tokenization, context windows, and common families (GPT, LLaMA, Mistral, Claude, Gemini).

Topics:
- How transformers work at a high level (attention, positional encoding)
- Tokenization and token budgets
- Model families and trade-offs (open vs closed source)
- Concepts: zero-shot, few-shot, fine-tuning, adapters, transfer learning

Hands-on:
- Explore Hugging Face Hub models
- Run inference with OpenAI, Hugging Face Inference API, or local runtimes (Ollama/llama.cpp)
