# AI Agents

Summary:
AI agents orchestrate tools and models to perform multi-step tasks. This includes tool use, state management, and planning.

Concepts:
- Tool calling and external API orchestration
- Planning algorithms and chains of thought for multi-step tasks
- Memory and context management for longer interactions
- Safety and sandboxing for tool execution

Hands-on:
- Build a simple agent that calls web APIs, uses a vector DB for memory, and chains steps to complete tasks
