# Prerequisites — Foundations for AI Engineering

Summary:
Build a strong foundation before deep AI work: programming, mathematics, and basic ML concepts.

Topics to cover:
- Programming: Python, version control (Git), package management
- Math: linear algebra, probability, statistics, optimization basics
- Machine learning fundamentals: supervised/unsupervised learning, evaluation metrics
- Data engineering basics: data cleaning, ETL pipelines, SQL

Practical exercises:
- Implement simple models with scikit-learn
- Explore datasets and build preprocessing pipelines
- Small projects: classification/regression end-to-end

Backlinks:
- Source roadmap: https://roadmap.sh/ai-engineer

Recommended pace: 4–8 weeks depending on prior experience.
