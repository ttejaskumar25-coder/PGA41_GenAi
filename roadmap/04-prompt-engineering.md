# Prompt Engineering

Summary:
Prompt engineering focuses on designing inputs to get desired outputs from LLMs. It's an experimental craft combining framing, examples, and constraints.

Techniques:
- Zero-shot vs few-shot prompting
- Chain-of-thought (CoT) and reasoning prompts
- Instruction tuning and system messages
- Prompt templates, conditioning, and output constraints (temperature, top-k)

Robustness:
- Test for prompt injection and adversarial inputs
- Add guardrails: input validation, output filtering, and safety prompts

Backlinks:
- Source roadmap: https://roadmap.sh/ai-engineer

Exercises:
- Create prompt templates for summarization, Q&A, and code generation
- Evaluate outputs with automated metrics and human review
